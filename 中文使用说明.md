# Nikon NP3/NCP 工具中文使用说明

本工具用于处理 Nikon Picture Control 文件，主要支持：

- 将 Photoshop/Lightroom 的 `.xmp` 预设转换为 Nikon 可导入的 `.NP3` 或 `.NCP` 文件。
- 将 `.NP3/.NCP` 文件导出为 Adobe 可读取的近似 `.xmp` 预设。
- 修复从网上下载后无法正常导入相机的 `.NP3/.NCP` 文件。
- 将转换或修复后的文件直接保存到 SD 卡的 `NIKON/CUSTOMPC` 文件夹。
- 浏览本地 Nikon recipe 预设，并查看近似预览效果。

> 注意：Nikon 的 `.NP3/.NCP` 是专有二进制格式。本工具基于已观察到的文件结构和模板修复方式实现，预览效果只是近似模拟，不等同于相机内的真实渲染。

## 1. 准备工作

### 安装 Python

请先安装 Windows 版 Python 3.10 或更新版本。

安装时建议勾选：

```text
Add python.exe to PATH
```

### 安装依赖

打开 PowerShell，进入项目的 `Picture Control` 文件夹：

```powershell
cd "项目路径\Picture Control"
```

然后运行：

```powershell
python -m pip install -r requirements.txt
```

依赖包括：

- `Pillow`：用于读取 JPG/PNG 预览图片。
- `PySide6`：用于中文桌面界面。

## 2. 启动中文桌面界面

推荐使用 PySide6 中文界面。

### 方法一：双击启动

进入 `Picture Control` 文件夹，双击：

```text
launch_pyside_converter.bat
```

### 方法二：命令启动

在 PowerShell 中运行：

```powershell
python pyside_converter.py
```

启动后窗口标题应显示为：

```text
Nikon NP3/NCP 工具
```

## 3. 界面区域说明

### 单个 XMP

用于选择一个 Photoshop/Lightroom `.xmp` 预设，并转换为 Nikon 文件。

- `选择 XMP`：选择一个 `.xmp` 文件。
- `预览 XMP`：查看转换后的近似效果。
- `导出 XMP 文件...`：将 XMP 转换为 `.NP3` 或 `.NCP` 文件并保存到电脑。
- `保存 XMP 到 SD 卡`：自动写入 SD 卡的下一个可用 `PICCON##` 文件名。

### 相机输出

用于设置输出格式和文件名。

- `格式`：选择 `NP3` 或 `NCP`。
- `下一个文件`：显示或编辑即将保存的 `PICCON##.NP3/.NCP` 文件名。

通常建议保持自动生成的 `PICCON##` 名称，方便相机识别。

### 预览图片

用于选择预览用的图片。

- `选择 JPG/PNG`：用自己的照片作为预览图。
- `使用生成图`：使用程序生成的默认示例场景。
- `向左旋转` / `向右旋转`：只旋转预览显示，不会修改原始图片。

### 工具

用于处理已有 Nikon 文件或浏览预设。

- `预制预设`：浏览本地 recipe 文件夹里的 `.NP3` 预设。
- `修复/导出`：选择一个 `.NP3/.NCP` 文件，修复后保存到电脑。
- `修复到 SD 卡`：选择一个 `.NP3/.NCP` 文件，修复后直接保存到 SD 卡。
- `NP3/NCP 转 XMP`：读取 `.NP3/.NCP` 中可识别的基础参数，并生成近似 `.xmp` 预设。

### 文件夹批量转换

用于批量转换一个文件夹里的 `.xmp` 文件。

- `选择输入文件夹`：选择包含 `.xmp` 文件的文件夹。
- `选择输出文件夹`：选择转换后文件保存的位置。
- `转换文件夹`：批量转换所有找到的 `.xmp` 文件。

## 4. 将 XMP 转换为 Nikon 文件

1. 启动中文桌面界面。
2. 在 `相机输出` 中选择输出格式：`NP3` 或 `NCP`。
3. 点击 `选择 XMP`，选择一个 `.xmp` 预设文件。
4. 右侧会显示近似预览。
5. 点击 `导出 XMP 文件...`。
6. 选择保存位置。
7. 完成后会弹出成功提示。

## 5. 直接保存到 SD 卡

1. 将相机 SD 卡插入电脑。
2. 启动工具。
3. 点击 `选择 XMP`，选择要转换的 `.xmp` 文件。
4. 点击 `保存 XMP 到 SD 卡`。
5. 如果程序找不到 SD 卡，会要求你选择 SD 卡根目录或 `NIKON/CUSTOMPC` 文件夹。
6. 工具会自动寻找 `PICCON01` 到 `PICCON99` 中第一个空位并保存。

SD 卡目标路径通常类似：

```text
E:\NIKON\CUSTOMPC
```

保存后，将 SD 卡插回相机，在相机的 Picture Control 导入菜单中导入即可。

## 6. 修复 NP3/NCP 文件

如果从网上下载的 `.NP3/.NCP` 文件无法被相机识别，可以尝试修复。

### 修复并保存到电脑

1. 点击 `修复/导出`。
2. 选择要修复的 `.NP3` 或 `.NCP` 文件。
3. 选择保存位置。
4. 工具会使用项目中的 `PICCON01.NP3` 作为模板重建文件。

### 修复并直接保存到 SD 卡

1. 插入 SD 卡。
2. 点击 `修复到 SD 卡`。
3. 选择要修复的 `.NP3` 或 `.NCP` 文件。
4. 工具会自动保存到 SD 卡的下一个空闲 `PICCON##` 槽位。

## 7. 浏览预制预设

点击 `预制预设` 后，工具会自动查找以下位置：

```text
recipes\
Nikon-Recipes-main\
..\Nikon-Recipes-main\
```

如果没有自动找到，会让你手动选择 Nikon Recipes 文件夹。

在预设浏览窗口中：

- 左侧选择预设。
- 右侧查看近似预览和参数。
- 点击 `保存到 SD 卡` 可直接修复并写入 SD 卡。
- 点击 `修复/导出...` 可保存到电脑。

## 8. 将 NP3/NCP 导出为近似 XMP

这个功能会读取 `.NP3/.NCP` 中能识别的基础 Picture Control 参数，并生成 Adobe Camera Raw / Lightroom 可读取的 `.xmp` 文件。

操作步骤：

1. 点击 `NP3/NCP 转 XMP`。
2. 选择一个 `.NP3` 或 `.NCP` 文件。
3. 选择 `.xmp` 保存位置。
4. 导出完成后，可以把 `.xmp` 放入 Lightroom 或 Adobe Camera Raw 的预设目录中测试。

当前会尝试反向映射这些字段：

```text
Nikon 对比度   -> XMP Contrast2012
Nikon 高光     -> XMP Highlights2012
Nikon 阴影     -> XMP Shadows2012
Nikon 白色色阶 -> XMP Whites2012
Nikon 黑色色阶 -> XMP Blacks2012
Nikon 饱和度   -> XMP Saturation
Nikon 清晰度   -> XMP Clarity2012
```

需要注意：

- 这是近似导出，不是还原原始 Lightroom 预设。
- Nikon 机内渲染和 Adobe 渲染不同，所以视觉效果不会完全一致。
- 曝光、白平衡、复杂曲线、色彩混合、色彩分级等信息可能无法完整反推。

## 9. 更换修复模板

项目文件夹内的 `PICCON01.NP3` 是修复时使用的模板文件。

为了提高与你自己相机的兼容性，建议：

1. 在相机中导出一个可以正常工作的 Picture Control。
2. 将导出的文件命名为：

```text
PICCON01.NP3
```

3. 替换 `Picture Control` 文件夹中的同名文件。

替换前可以先备份原文件。

## 10. 常见问题

### 双击 bat 没反应

请先确认已经安装 Python，并且运行过：

```powershell
python -m pip install -r requirements.txt
```

如果仍然失败，可以在 PowerShell 中运行：

```powershell
python pyside_converter.py
```

这样可以看到具体错误信息。

### 提示缺少 PySide6

进入 `Picture Control` 文件夹后运行：

```powershell
python -m pip install -r requirements.txt
```

### 提示缺少 Pillow

同样运行：

```powershell
python -m pip install -r requirements.txt
```

### 找不到 SD 卡

可以手动选择：

- SD 卡根目录，例如 `E:\`
- 或 `E:\NIKON\CUSTOMPC`

如果 SD 卡里没有 `NIKON/CUSTOMPC`，工具会尝试自动创建。

### 相机仍然无法识别

可以尝试：

1. 确认文件名是 `PICCON##.NP3` 或 `PICCON##.NCP`。
2. 确认文件放在 SD 卡的 `NIKON/CUSTOMPC` 文件夹。
3. 使用自己相机导出的 `PICCON01.NP3` 替换项目模板。
4. 尝试另一个空闲编号，例如 `PICCON02.NP3`。

## 11. 命令行用法

如果不使用桌面界面，也可以用命令行。

转换单个 XMP：

```powershell
python photo_preset_to_nikon.py --input path\to\preset.xmp --output path\to\output.np3
```

批量转换文件夹：

```powershell
python photo_preset_to_nikon.py --input-folder path\to\presets --output-folder path\to\nikon-profiles
```

将 NP3/NCP 导出为近似 XMP：

```powershell
python photo_preset_to_nikon.py --export-xmp --input path\to\profile.np3 --output path\to\preset.xmp
```

## 12. 推荐使用流程

第一次使用时，建议按这个顺序：

1. 安装依赖。
2. 双击 `launch_pyside_converter.bat`。
3. 选择一个 XMP 文件。
4. 先用右侧预览确认大致效果。
5. 导出到电脑测试。
6. 确认没问题后再保存到 SD 卡。
7. 如果相机无法识别，换用自己相机导出的 `PICCON01.NP3` 模板再试。
